# People and Process

## The first contacts

- Manager: priorities and expectations.
- Buddy: practical day-to-day onboarding help.
- Tech lead: architecture and engineering workflow questions.
- Support owner: workflow issues and customer-facing escalations.

## Process notes

- Ask for the latest version of a document rather than relying on copies.
- Capture decisions in docs so future new joiners do not need to reverse engineer context.
- Keep operational changes visible.
